## 树
树的本质是递归。

### 树的遍历
- 广度遍历
- 深度遍历
    - 先序遍历：先遍历根节点，即**根->左->右**
    - 中序遍历：根节点在中间，即**左->根->右**
    - 后序遍历：根节点在后面，即**左->右->根**
